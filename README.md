## 开发进程⛽️😊

### 注意!
* 最好在本地装好Github客户端，方便查看修改的代码文件。
* 点击Download，下载最新版代码。
* 修改前最好运行程序(`CatchTigerChess.java`)来检查程序能否正常运行。
* 最好不修改已写好的代码。
* 写完了相应的代码(在本地测试无误),在 **experiment** 分支上传代码(有注释最好啦~)。
